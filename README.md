# MinecraftServerRunner
A website that will be a main host for running minecraft servers with ease.

Running the app:
To run the app you will need to have python 3.8 as this is the version it was originally developed in. Additionally, any python enviroment capable of running the python version.

Future features:
Hopefully eventually have the ability to add server folders to a database and having the freedom to pick and choose different list of minecraft servers all in one place.
